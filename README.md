# taskas
A simple task manager app intend to run in your browser.

## How to run
1. Simply clone the repo

```
git clone https://github.com/saifshahriar/taskas.git --depth=1
```

2. And run the `index.html` file in your browser.

```
xdg-open ./index.html
```
